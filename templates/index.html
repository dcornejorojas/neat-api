Neat's Pokemon adoption API

Endpoints Usage

  • [GET] /health: health endpoint. Retrieve status of the API
  • [GET] /pokemon/list Retrieve list of unnadopted pokemons
  • [POST] /pokemon/healthcare: Retrieve updated info for those pokemon that with nurse Joy
  • [GET] /trainer/status: Retrieve information by a given adoption process
  • [POST] /trainer/adoption: Process a new pokemon adoption followed by different metrics.
